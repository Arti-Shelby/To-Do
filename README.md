# To-Do List

A simple local to-do list app. Add tasks, check them off, delete them — that's it.

## Stack

- HTML
- SCSS
- Vanilla JS

## Getting Started

Clone the repo and open `index.html` in a browser. No build step, no dependencies.

```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
open index.html
```

If you want to compile SCSS:

```bash
sass style.scss style.css
```

## Features

- Add tasks
- Mark tasks as completed
- Delete tasks
- Data saved in localStorage
